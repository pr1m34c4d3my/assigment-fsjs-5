## How To Run

This is created with HTML, Tailwind CSS, and little touch by GSAP JS
To run this first,

`npm install`

run that code to install node_modules into your directory.

After that, run this code to --watch compile the css of tailwind automaticly

`npm run dev`
or
`yarn dev`
